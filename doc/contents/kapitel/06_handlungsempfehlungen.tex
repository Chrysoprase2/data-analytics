\chapter{Ableitbare Handlungsempfehlunge}
\label{chap:handlungsempfehlungen}

\subsubsection{Ableitbare Handlungsempfehlungen}
Basierend auf den Analyseergebnissen lassen sich folgende Empfehlungen für eine fiktive Sozialpolitik der damaligen Zeit formulieren:
\begin{itemize}
    \item \textbf{Stabilisierung prekärer Arbeitsverhältnisse:} Da ungelernte Tagelöhner das höchste Risiko trugen, wäre eine Transformation hin zu regulären Beschäftigungsverhältnissen eine effektive Maßnahme gewesen.
    \item \textbf{Gezielte Unterstützung:} Da Haushalte mit weiblichen Vorständen eine signifikant höhere Armutsrate aufwiesen, hätte es spezifischer Unterstützungssysteme bedarft.
    \item \textbf{Strukturelle statt demografische Fokussetzung:} Maßnahmen sollten eher auf Arbeitsmarktreformen und Mindestlöhne abzielen als auf die Kontrolle der Familiengröße, da letztere statistisch einen geringeren Erklärungsbeitrag zur Armut leistete.
\end{itemize}

\subsubsection{Fazit und Reflexion}
Dieses Mini-Projekt demonstriert einen Nutzen moderner statistischer Verfahren für die Interpretation historischer Daten. Trotz der Einschränkung, dass die Daten ursprünglich nicht für die moderne Analyse erhoben wurden und keine Normalverteilung aufwiesen, konnten durch aktuelle Verfahren belastbare Aussagen getroffen werden. Der Prozess von der manuellen Datenerfassung bis zur Modellbildung verdeutlicht, dass eine kritische Prüfung der Vorbedingungen essenziell ist, um Fehlinterpretationen zu vermeiden.
